# Project Guardrails for Claude Code

This is a learning project. Please follow these rules:

1. **Explain before big changes** — Before making significant changes to the code or project structure, briefly explain what you're about to do and why.

2. **Ask before deleting files** — Never delete a file without asking first and explaining why.

3. **Keep the code simple** — I'm learning. Prefer simple, readable code over clever or advanced patterns, even if a more complex approach would be more efficient.

4. **No build frameworks** — This should stay plain HTML, CSS, and JavaScript. Do not introduce React, Vue, webpack, or any other build tooling or framework.

5. **No backend, no database** — This is a static website. Don't add a server, API backend, or database.
